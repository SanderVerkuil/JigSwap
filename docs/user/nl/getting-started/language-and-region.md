# Taalinstellingen (Engels / Nederlands)

JigSwap is beschikbaar in het **Engels (en)** en het **Nederlands (nl)**.

1. Zoek de **taalwisselaar** in de app.
2. Selecteer je voorkeurstaal.
3. De interface (en de inlogschermen) wisselt direct, en je keuze wordt onthouden voor volgende bezoeken.

JigSwap kiest automatisch een begintaal op basis van je opgeslagen voorkeur, daarna de taal van je browser, met Engels als standaard.
