# Snel Zoeken (Opdrachtenpalet)

JigSwap heeft een globale zoekfunctie waarmee je snel alles vindt.

- **Openen:** druk op **Cmd/Ctrl + K**, of klik op de **zoekpil** in de bovenbalk op desktop.
- **Met niets ingetypt:** je ziet navigatiebestemmingen om naartoe te springen en snelle-actie-snelkoppelingen.
- **Type 2 of meer tekens:** live resultaten verschijnen, gegroepeerd in **Puzzels**, **Mensen**, **Kringen** en **Collecties**.
- **Selecteer een resultaat** om er direct naartoe te navigeren.
