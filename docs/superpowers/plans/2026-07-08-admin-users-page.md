# Admin Users Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** PR 2 of the approved spec (`docs/superpowers/specs/2026-07-08-admin-users-puzzles-and-confirm-dialogs-design.md`): a read-only admin page at `/admin/users` listing every member (avatar+name, username, email, joined date, active flag, admin badge, owned-copy count) with search and load-more pagination. To render a role column, the Clerk `publicMetadata.role` is mirrored onto the `users` table (webhook sync + one-shot backfill action). **The mirrored field is display-only — authorization stays JWT-based via `identity/isAdmin` and must never read `users.role`.**

**Architecture:** Follows the existing thin-admin-read-model precedent (`admin/getModerationActivity`, `admin/listRejectedPhotos`): a Convex query gated by `requireMember` + `isAdmin`, returning a typed `@jigswap/contracts` view DTO, exposed through the `@jigswap/gateway` `admin` namespace. No domain-package involvement (pure read). Frontend is a new TanStack Router leaf inside the existing `/admin` gate layout, using the sanctioned `usePaginatedQuery` exception (same as `puzzle-client.tsx`). Role mirroring extends the existing Clerk webhook sync (`users.updateOrCreateUser`) and copies the `backfillSearchableName` backfill pattern for pre-existing rows (via the Clerk Backend API, since the role lives in Clerk, not Convex).

**Tech Stack:** Convex (queries/mutations/actions, `convex-test` + vitest for backend tests), Clerk (`@clerk/backend` ^3.x), TanStack Router/Query + `usePaginatedQuery`, shadcn UI primitives, use-intl (en + nl), Nx monorepo (pnpm).

```ts
"use node";

import { createClerkClient } from "@clerk/backend";
import { v } from "convex/values";
import { internal } from "./_generated/api";
import { internalAction } from "./_generated/server";

// One-shot backfill for the mirrored `users.role` display field (pattern: backfillSearchableName,
// except the source of truth is Clerk, so this pages the Clerk Backend API instead of the table).
// Patches EVERY member's mirror by clerkId — including clearing rows whose Clerk role was removed.
// Internal-only; run once from the Convex dashboard/CLI after deploy:
//   npx convex run backfillUserRoles:run '{}'
// Rows written after deploy stay current via the user webhook (users.updateOrCreateUser).
export const run = internalAction({
  args: { pageSize: v.optional(v.number()) },
  handler: async (ctx, { pageSize }) => {
    const clerkClient = createClerkClient({
      secretKey: process.env.CLERK_SECRET_KEY,
    });

    const limit = pageSize ?? 100;
    let offset = 0;
    let scanned = 0;

    // Clerk's list endpoint is offset-paginated; totalCount says when to stop.
    for (;;) {
      const page = await clerkClient.users.getUserList({ limit, offset });
      for (const clerkUser of page.data) {
        const role = (
          clerkUser.publicMetadata as Record<string, unknown> | null | undefined
        )?.role;
        await ctx.runMutation(internal.users.patchUserRole, {
          clerkId: clerkUser.id,
          role: typeof role === "string" ? role : undefined,
        });
      }
      scanned += page.data.length;
      offset += page.data.length;
      if (page.data.length < limit || scanned >= page.totalCount) break;
    }

    return { scanned };
  },
});
```

```ts
// Admin read model: every member, paginated newest-first, with the display-only mirrored Clerk

// getModerationActivity. An optional search term switches to the by_searchable_name search index

      .query("ownedPuzzles")
      .withIndex("by_owner", (q) => q.eq("ownerId", user._id))
      .collect()
  ).length,
  createdAt: user.createdAt,
});

export const listUsers = query({
  args: {
    paginationOpts: paginationOptsValidator,
    search: v.optional(v.string()),
  },
  handler: async (ctx, args): Promise<PaginationResult<AdminUserView>> => {
    await requireMember(ctx);
    if (!(await isAdmin(ctx))) throw new ConvexError("Forbidden");

    const search = args.search?.trim().toLowerCase() ?? "";
    const result =
      search.length > 0
        ? await ctx.db
            .query("users")
            .withSearchIndex("by_searchable_name", (q) =>
              q.search("searchableName", search),
            )
            .paginate(args.paginationOpts)
        : await ctx.db
            .query("users")
            .order("desc")
            .paginate(args.paginationOpts);

    return {
      ...result,
      page: await Promise.all(result.page.map((u) => toAdminUserView(ctx, u))),
    };
  },
});
```
